
## Deployment
Ouvrir 2 terminaux

To deploy this project run

1 terminal dans le dossier back
```bash
  npm install
```
```bash
  npm run build
```
```bash
  npm start
```
1 terminal dans le dossier front

```bash
  npm install
```
```bash
  npm start
```
To launch the tests dans le dossier back

```bash
  npm test
```
